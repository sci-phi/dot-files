# dot-files

Basic "dot" files for shell profile customization.

## Bootstrapping a new system.

The goal is to build up a working system from 'scratch', via the simplest common entry-point, the shell environment.

Refer to [Bootstraping](docs/bootstrapping.md) for details.

On the terminal command-line, CD to 'dot-files' directory and execute : `bin/bootstrap.sh`

This will install [Homebrew](https://homebrew.sh), if necessary, and also the tools specified in the adjacent `Brewfile`
